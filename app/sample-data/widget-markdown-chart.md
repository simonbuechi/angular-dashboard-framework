# Charts

Based on D3 and C3. See [http://c3js.org/examples.html](http://c3js.org/examples.html) for examples. 

## Sample widget configuration

```javascript
{
  type: "chart",
  title: "Spline chart",
  config: { 
    url: "data/json-data-chart.json",
    type: "spline",
    height: "160",
    showLegend: false,
    showTooltip: true,
    rotated: false
  }
}
```
